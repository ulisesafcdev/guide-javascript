Generadores

Una función generadora, es un tipo especial de función que sirve como una fabrica de iteradores. Cuando se ejecuta, regresa un nuevo objeto Generador. Una función se convierte en una función generadora si utiliza la sintaxis function*

Creando un nuevo generador: 

                
function* mensajes() {
    yield "Mensaje #1"
    console.log("Otro mensaje por separado...")
    yield "Mensaje #2"
    yield "Mensaje #3"
    console.log("Ultimo mensaje por separado...")
}

Para crear una función generadora, luego de definir la palabra function, justo a la par escribimos un asterisco *, para hacer saber al programa que es una función generadora. Dentro de la función generadora, automáticamente activa un iterador, para ello, usamos la palabra reservada yield, que solamente se puede usar dentro de una función generadora. Al usar esta palabra, llama al método .next() con las dos propiedades llamadas "value" & "done".

yield nos permite pausar y reanudar algo iterable. Una vez definido que es lo que vamos a iterar dentro de la función generadora, veamos como podemos mostrarlo en consola con un bucle: 

                
let iterador = mensajes()

for (const i of iterador) {
    console.log(i)
}

En una variable almacenamos la función generadora, es decir, al ser una función generadora, lo que almacenamos en la variable es el iterador. Dentro del bucle automáticamente se muestran los mensajes de la función.
