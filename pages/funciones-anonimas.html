#funciones-anónimas-autoejecutables

Son conocidas como funciones anónimas autoejecutables, son funciones que se ejecutan tan pronto como se definen.

IIFE: Expresión de función ejecutada inmediatamente

Las expresiones de función ejecutadas inmediatamente (IIFE por su sigla en ingles), es un patron de diseño también conocido como función autoejecutable y se compone por dos partes. La primera es la función anónima con alcance léxico encerrado por el operador de agrupación (). Esto impide accesar variables fuera del IIFE, asi como contaminar el alcance (scope) global.

La segunda parte crea la expresión de función cuya ejecución es inmediata (), siendo interpretado directamente en el engine de JavaScript.

Sintaxis

Podemos crear una función autoejecutable en forma de función flecha: 

                
(() => {

})()

Podemos crear una función autoejecutable en forma de función anónima: 

                
(function() {

})()

En ambos casos podemos mandar parámetros, veamos un ejemplo en forma de función flecha: 

                
((nombre) => {
    console.log(`Bienvenido ${nombre}`)
})("Ulises Flores");

Veamos un ejemplo mandando parámetros en forma de función anónima: 

                
(function(usuario) {
    console.log(`Hola y bienvenido ${usuario}`)
})("ulisesafcdev");
