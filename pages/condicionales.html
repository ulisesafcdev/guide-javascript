#condicionales

Ejecuta una sentencia si una condición especificada es evaluada como verdadera. Si la condición es evaluada como falsa, otra sentencia puede ser ejecutada.

Ejemplo

Veremos un ejemplo calculando la edad de un usuario y determinar si es mayor de edad o no: 

                
let edad = 24

if (edad >= 18) {
    console.log(`Eres mayor de edad`)
} else {
    console.log(`Aun eres menor de edad`)
}

En el ejemplo anterior utilizamos la sentencia condicional if-else. Definimos una variable llamada edad con el valor de 24, con la sentencia if evaluamos si la condición se cumple, si la edad es mayor o igual a 18 imprimir en pantalla que eres mayor de edad, si esta condición no se cumple, la condición entra en else.

Multiples sentencias

Multiples sentencias if-else pueden ser anidadas para crear una clausula else-if. Veamos el mismo ejemplo anterior: 

                
let edad = 84

/* if (edad >= 18) {
    console.log(`Eres mayor de edad`)
} else {
    console.log(`Aun eres menor de edad`)
} */

if (edad < 18) {
    console.log(`Aun eres menor de edad`)
} else if (edad >= 18 && edad < 60){
    console.log(`Eres mayor de edad`)
} else {
    console.log(`Eres un adulto mayor de la tercera edad`)
}

Un ejemplo sencillo de multiples sentencias, con el mismo ejemplo calculamos las posibles respuestas, evaluamos en caso de que el usuario sea menor de edad, evaluamos en caso de que sea mayor de edad y que no sobrepase la tercera edad, y finalmente si no se cumplen las sentencias if, imprimir un mensaje.

La condición que se cumpla es la que entrara en la ejecución del programa.
