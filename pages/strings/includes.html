.includes()

El metodo .includes() determina si una cadena de texto puede ser encontrada dentro de otra cadena de texto, devolviendo true o false segun corresponda.

Este metodo permite determinar si una cadena de texto se encuentra incluida dentro de la otra. El metodo .includes() es case sensitive tiene en cuenta mayusculas y minusculas.

Veamos un ejemplo: 

                
let str = "Teoria y practica en JavaScript"
let word = "practica"

console.log(`La palabra "${word}" ${str.includes(word) ? "SI" : "NO"} se encuentra en la cadena.`)

Definimos una variable con un texto, y otra con la palabra a evaluar. Hacemos uso del metodo .includes() y le pasamos como parametro la palabra a evaluar, hacemos uso del operador ternario para evaluar en caso de que nos devuelva true usar la palabra SI y concatenar con la oracion.
