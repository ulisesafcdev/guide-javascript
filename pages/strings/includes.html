.includes()

El método .includes() determina si una cadena de texto puede ser encontrada dentro de otra cadena de texto, devolviendo true o false según corresponda.

Este método permite determinar si una cadena de texto se encuentra incluida dentro de la otra. El método .includes() es case sensitive tiene en cuenta mayúsculas y minúsculas.

Veamos un ejemplo: 

                
let str = "Teoria y practica en JavaScript"
let word = "practica"

console.log(`La palabra "${word}" ${str.includes(word) ? "SI" : "NO"} se encuentra en la cadena.`)

Definimos una variable con un texto, y otra con la palabra a evaluar. Hacemos uso del método .includes() y le pasamos como parámetro la palabra a evaluar, hacemos uso del operador ternario para evaluar en caso de que nos devuelva true usar la palabra SI y concatenar con la oración.
