#objetos

La clase Object representa uno de los tipos de datos en JavaScript. Es usado para guardar una colección de datos definidos y entidades mas complejas. Los objetos pueden ser creados utilizando el constructor o la sintaxis literal.

Creación de un objeto

Veamos como podemos crear un objeto: 

                
const userData = {
    nombres: "Ulises Alexis",
    apellidos: "Flores Chavez",
    edad: 24,
    lenguajes: ["html5", "css3", "javascript", "php"],
    residencia: {
        pais: "El Salvador",
        departamento: "San Salvador",
        municipio: "Apopa"
    },
    estudiante: true,
    mensaje: function () {
        console.log('hello world')
    }
}

Para crear un objeto usaremos la palabra const para definir el nombre de nuestro objeto, como valor pondremos unas llaves {} y dentro de estas se van a clasificar como llave: valor separados por comas (,) cada valor admite cualquier tipo de dato primitivo e incluso arreglos, objetos, y funciones.
