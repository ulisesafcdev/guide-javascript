#funciones-flecha

Una expresión de función flecha es una alternativa compacta a una expresión de función tradicional. 

                
const bienvenida = () => {
    console.log(`Bienvenido...`)
}

bienvenida()

Su sintaxis es similar a la de una expresión de función, omitiendo la palabra reservada function y agregando el símbolo => después de los paréntesis.

Una de las ventajas que nos permiten usar funciones flecha es poder compactar nuestro código. Veamos un ejemplo: 

                
const suma = (a, b) => { return a + b }

let resultado = suma(12, 8)
console.log(resultado)
