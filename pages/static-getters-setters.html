#static-getters-setters

Son utilizados en métodos dentro de una clase, son parte de un nuevo set de azúcar sintáctico que vinieron en ECMAScript 2015.

#static

La palabra clave static define un método estático para una clase.

Los métodos estáticos son llamados sin instanciar su clase. Son habitualmente utilizados para crear funciones para una aplicación. 

                
class Suma {

    static sumar(a, b) {
        return a + b
    }

}

let resultadoSuma = Suma.sumar(12, 8)

console.log(`Resultado: ${resultadoSuma}`)

#get & #set

Los métodos get & set nos permiten establecer y obtener los valores de atributos de una clase. 

                
class Perro {

    constructor(nombre, color) {
        this.nombre = nombre
        this.color = color
        this.raza = null
    }

    imprimir() {
        console.log(`Nombre del perro: ${this.nombre}`)
        console.log(`Color del perro: ${this.color}`)
    }

    // metodo setter
    set setRaza(raza) {
        this.raza = raza
    }

    // metodo getter
    get getRaza() {
        return this.raza
    }
}

En el ejemplo anterior, hemos definido métodos get y set con los nombres setRaza() y getRaza().

Para establecer un valor con set se hace de la siguiente manera: 

                
const cabito = new Perro("Cabito", "Blanco y Negro")

cabito.setRaza = "Pastor Alemán"

Primero hacemos una instancia de la clase, luego para establecer un valor con set le asignamos un valor con el símbolo =, asi como asignamos el valor a una variable.

Para obtener un valor con get se hace de la siguiente manera: 

                
let raza = cabito.getRaza

El método get nos retorna un valor, por eso creamos una variable para almacenarlo ahi. Para invocar al metodo get se hace con la notación del punto, pero, sin los paréntesis ().
