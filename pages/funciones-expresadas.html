#funciones-expresadas

Las funciones también se pueden crear mediante una expresión de función. Esta función puede ser anónima; no tiene porque tener un nombre.

Definiendo una función expresada

                
const cuadrado = function (num) {

    if (typeof num == "undefined") return console.warn("Debes ingresar un valor numerico.");
    if (typeof num !== "number") return console.warn("Ingresa un numero para evaluar.");

    return num * num
}

let resultado = cuadrado(3)
console.log(resultado)

Para definir una función expresada hacemos uso de const seguido de un nombre como si fuéramos a declarar una variable, pero el valor de esta sera la función expresada. Dentro podemos hacer el procedimiento que queremos que haga como cualquier función, lo único que cambia es la forma en como las definimos.

Dentro de la función expresada estamos recibiendo un parámetro, un valor numérico para elevarlo al cuadrado. Se hacen las validaciones en diferentes casos que pueden dar errores cuando el usuario intente interactuar con el programa. Finalmente retornamos un valor elevado al cuadrado, lo guardamos en una variable resultado para imprimirlo en consola.
