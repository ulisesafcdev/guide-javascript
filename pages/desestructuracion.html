#desestructuracion

La sintaxis de desestructuracion es una expresión de JavaScript que permite desempacar valores de arreglos o propiedades de objetos en distintas variables.

Desestructuracion de un arreglo

                
const notasAlumno = [7,8,5,9,5]

console.log(notasAlumno)

Tenemos un arreglo llamado "notasAlumno" que almacena en cada posición una nota correspondiente a una materia en especifico. Pero, que pasaría si nosotros queremos que cada posición de nuestro arreglo de notas se almacene en una variable con el nombre de la materia a la que corresponde?

Veamos como se puede hacer con la sintaxis de desestructuracion: 

                
const [matematicas, lenguaje, ciencia, sociales, ingles] = notasAlumno

console.log(`Matematicas ${matematicas}\nLenguaje: ${lenguaje}\nCiencia: ${ciencia}\nSociales: ${sociales}\nIngles: ${ingles}`)

Definimos en forma de arreglo con los nombres de la materia según la posición a la que queremos relacionar. Definiendo como valor igual al arreglo llamado "notasAlumno". Una vez hecho esto ya podemos acceder a estas variables por separado con el nombre que le hemos asignado y mostrarlas en consola con el nombre de la materia que corresponde.

Desestructuracion de un objeto

                
let persona = {
    nombres: "Ulises Alexis",
    apellidos: "Flores Chavez",
    edad: 24,
    trabaja: true
}

console.log(persona)

Tenemos un objeto llamado "persona" con diferentes llaves y valores. Al igual que el ejemplo anterior, que pasaría si queremos almacenar cada llave con su valor en una variable correspondiente a cada una?

Veamos como se puede hacer con la sintaxis de desestructuracion:

                
let {nombres, apellidos, edad, trabaja} = persona

console.log(`Nombres: ${nombres}\nApellidos: ${apellidos}\nEdad: ${edad}\nTrabaja: ${trabaja}`)

Definimos en forma de objeto los nombres de las variables correspondientes a cada llave y valor de nuestro objeto que estamos desestructurando. Una vez definidos los nombres, ya podemos acceder a ellos y poderlos mostrar en consola teniendo como resultado lo siguiente:

Hay que tener en cuenta que a la hora de desestructurar un objeto, los nombres de las variables deben ser igual a las llaves del objeto, de lo contrario nos mostrara undefined cuando queramos mostrar el valor. 

                
let persona = {
    nombres: "Ulises Alexis",
    apellidos: "Flores Chavez",
    edad: 24,
    trabaja: true
}

let {nombres, apellidos, edad, empleo} = persona

console.log(`Nombres: ${nombres}\nApellidos: ${apellidos}\nEdad: ${edad}\nTrabaja: ${empleo}`)

En el código tenemos nuestro objeto con la llave llamada "trabaja", pero, en la desestructuracion hemos definido con el nombre de "empleo", a la hora de mostrar en pantalla y acceder al valor de este a simple vista no ha pasado nada, pero al ver el resultado en consola nos muestra lo siguiente:

Solamente cuando se desestructura un objeto, las variables deben ser el mismo nombre que las llaves del objeto.
