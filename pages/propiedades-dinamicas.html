Propiedades dinámicas de los objetos

Hay una manera de crear propiedades dinamicamente en objetos, por ejemplo, si queremos crear una propiedad que identifique a cada usuario, veamos un ejemplo pequeño para ponerlo en practica:

Primero creemos un array con nombres y luego un objeto vacío. 

                
let usuarios = ["ulises", "juan", "mario", "oscar", "kevin", "luis", "melvin", "carlos", "toño", "messi"]

let objUser = { }

Para definir una propiedad que sea dinámica, nos basaremos en el array con los nombres almacenados. Crearemos una propiedad llamada "id_" seguido del numero del indice del registro, y su valor sera pues el valor de cada indice del array. 

                
usuarios.forEach((user, index) => {
    objUser[`id_${index}`] = user
})

console.log(objUser)

Primero, con un bucle recorremos el array de datos, de este array sacaremos su valor y su indice. Segundo, ya que tenemos el objeto vacío, para crear una propiedad se puede hacer llamando al objeto directamente y asignar un valor, pero, en este caso haremos una propiedad dinámica, y para crearla hacemos uso del corchete. Lo que estamos haciendo es que por cada valor del array se genero una propiedad llamada "id_0", "id_1", "id_2" ... y asi sucesivamente, y su valor sera pues el nombre de cada elemento del array.
